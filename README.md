
# 🚀 GitHub Project Analyzer AI

An AI-powered web application that analyzes GitHub repositories and evaluates them like a recruiter.

🔗 **Live Demo:**
https://code-reviewer--nvivek65.replit.app

---

## ✨ Features

* 🔍 Analyze any public GitHub repository
* 🧠 AI-powered code quality evaluation
* 📘 README completeness scoring
* 🛠 Tech stack detection
* 📊 Recruiter-style scoring system
* 🚀 Actionable improvement suggestions

---

## 🧠 How It Works

1. Enter a GitHub repository link
2. The system extracts repository data
3. AI analyzes:

   * Code quality
   * Documentation
   * Structure
4. Generates a recruiter-style evaluation
